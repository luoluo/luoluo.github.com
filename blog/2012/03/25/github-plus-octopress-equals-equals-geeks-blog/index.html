1.无法装ruby，问题如下

no such file to load – zlib (LoadError) 解决办法：

1.rvm package install zlib 
2.rvm remove 1.9.1
3.rvm install 1.9.1 -C --with-zlib-dir=$rvm

卡了半天最后还是十分感谢互联网的Hackers，让我从泥潭中巴腿而出。解决方案就是修复openssl:

1.sudo apt-get install libssl-dev 
2.cd ext/openssl 
  Run the command one after the others. 
  ruby extconf.rb 
  make 
  sudo make install

4.octopress的个性化配置

Main Configs
1.url: http://yoursite.com  # blog地址
2.title: My Octopress Blog  # blog名称，会出现在左上角 
3.subtitle: A blogging framework # title下面的副标题，可以用来放你的喜欢的名人名言，或者简单介绍下自己
4.author: Your Name # 会出现在每篇文章下面的Posted by xxx
5.simple search: http://google.com/search # 右边搜索框使用的搜索引擎
6.description: # head.html中meta name="description"的默认content，但一般都被会截断的文章内容替换掉

Jekyll &amp; Plugins
1.root: / # 网站的根目录，如果你网站是http://site.com/project这样的形式，就要写成 root: /project
#生成文章的url形式，比如不喜欢url中带时间，习惯后缀加html，可以写成 /blog/:title.html
2.permalink: /blog/:year/:month/:day/:title/
3.paginate: 10          # 每页展现blog的数量，如果blog数量大于这个，会产生翻页
4.recentposts: 5       # 右侧最近发表模块里blog的数量
  #在某篇blog中使用<!-- more -->，列表页就不会展现该blog全文，就会显示这个read on链接，指向blog全文
5.excerptlink: "Read on &rarr;"
  #右侧边栏的展现，有些要下面的设置配合，比如你没有配置githubuser，则这儿也不会展现github.html
  #也可以增加一些自己的页面，比如在/source/includes/custom/asides/下增加about.html
  #然后在列表中增加custom/asides/about.html即可，这儿前后的顺序，就代表了页面上下的次序 
6.defaultasides: [asides/recentposts.html, asides/github.html, asides/twitter.html]
# 可以给不同的页面配置不同的侧边栏  
7.blogindexasides:
8.postasides:
9.pageasides:
